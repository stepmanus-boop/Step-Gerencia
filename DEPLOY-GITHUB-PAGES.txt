GitHub Pages - publicação rápida

Este pacote foi ajustado para funcionar no GitHub Pages.

Opção 1 (recomendada para o seu repositório atual):
- Settings > Pages
- Source: Deploy from a branch
- Branch: main
- Folder: /(root)

Arquivos prontos na raiz:
- index.html
- app.css
- app.js
- projects.json
- assets/

Se preferir, também existe a pasta docs/ pronta.

Importante:
- No GitHub Pages, o painel usa projects.json estático.
- No Netlify, ele ainda tenta /api/projects primeiro e usa projects.json como fallback.
